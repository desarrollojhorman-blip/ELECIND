# Colores de botones

Componente: `<x-ui.button variant="...">`. Fondo blanco/transparente en hover cuando aplica.

| Variante    | Color aprox.        | Fondo normal     | Hover            | Cuándo usarlo                         |
|-------------|---------------------|------------------|------------------|---------------------------------------|
| `success`   | Verde esmeralda     | `emerald-600`    | `emerald-700`    | Guardar, confirmar, crear             |
| `danger`    | Rojo                | `red-600`        | `red-700`        | Eliminar, acción destructiva          |
| `info`      | Azul                | `blue-600`       | `blue-700`       | Editar, ver detalle, navegar          |
| `primary`   | Rojo oscuro (marca) | `primary-600`    | `primary-700`    | Acción principal del color de empresa |
| `warning`   | Ámbar / naranja     | `amber-500`      | `amber-600`      | Advertencia, acción con precaución    |
| `secondary` | Gris carbón         | `#1f2937`        | `#111827`        | Acción secundaria oscura              |
| `ghost`     | Sin color           | transparente     | `slate-100`      | Volver, cancelar, acciones neutras    |
| `outline`   | Blanco con borde    | `white`          | `slate-50`       | Alternativa a ghost con borde visible |
| `link`      | Solo texto          | transparente     | —                | Texto clicable inline, sin caja       |

---

## Referencia rápida: color → botón

| Color            | Botones que lo usan                          |
|------------------|----------------------------------------------|
| Verde esmeralda  | Guardar, Guardar cambios, Crear              |
| Rojo             | Eliminar                                     |
| Azul             | Editar, Nuevo, Ver                           |
| Rojo oscuro      | Acciones principales de marca                |
| Ámbar / naranja  | Advertencias                                 |
| Gris carbón      | Secundario (móvil: Aceptar)                  |
| Sin color        | Volver, Cancelar, Cerrar                     |
| Blanco + borde   | Acciones neutras con borde visible           |
| Solo texto       | Links inline                                 |
