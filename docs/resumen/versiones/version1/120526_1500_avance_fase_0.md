## 🚀 Rutas Disponibles

### Autenticación
- `GET /login` → Formulario de login
- `POST /login` → Procesar login (username + password)
- `POST /logout` → Cerrar sesión

### Web (Requiere auth + acceso 'web' o 'ambos')
- `GET /` → Panel web ELECIND (dashboard)

### Mobile (Requiere auth + acceso 'mobile' o 'ambos')
- `GET /m` → Panel móvil ELECIND (dashboard)

### API
- `GET /api/user` → Datos del usuario autenticado (Sanctum)

### Fortify (Default, no customizado)
- `GET /register` → Registro (deshabilitado por default)
- `GET /forgot-password` → Reset de contraseña
- `POST /reset-password` → Procesar reset
- `GET /email/verify` → Verificación de email

## 🔑 Decisiones Técnicas

### ❌ Evitadas
- ~~Tenancy multi-tenant~~ → Single-tenant simplificado
- ~~Fortify login por email~~ → Implementado con username
- ~~PHP 8.3 forcing~~ → Compatible con PHP 8.2.12 existente

### ✅ Adoptadas
- LoginController custom → Más control y flexibilidad
- Middleware de acceso explícito → Seguridad clara por rol
- Spatie Permission v6.25 → Compatible con PHP 8.2
- Maatwebsite Excel v3.1.69 → Modern + PHP 8.2 compatible
- Pint + Larastan en CI → Code quality automático
